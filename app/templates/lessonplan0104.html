Learning outcomes

After this lesson, participants will be able to:

  • Describe the mission of three social justice organizations and the roles of one staff person at each
  • Write "user stories" for a digital social justice tool that we have already thought about
  • Set up a GitHub account, make a repository, and explain what they both are to a non-coder
  • Push a document (our "user stories") to Github
